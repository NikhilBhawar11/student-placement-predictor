# Student Placement Prediction Web App

This project predicts whether a student will be placed based on academic performance and skills.

## Features
- CGPA
- Internships
- Projects
- Workshops/Certifications
- Aptitude Test Score
- Soft Skills Rating
- Extracurricular Activities
- Placement Training
- SSC Marks
- HSC Marks

## Tech Stack
Python, Flask, Scikit-Learn, Pandas, HTML, CSS

## Machine Learning Model
Logistic Regression Classifier

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run application

python app.py

3. Open browser

http://127.0.0.1:5000
