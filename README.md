# 🎓 Student Placement Prediction System

A Machine Learning web application that predicts whether a student will be placed based on academic and skill-based features.

---

## 🚀 Project Overview

This project predicts whether a student will get placed based on different academic and skill-related features such as:

- CGPA
- Internships
- Projects
- Workshops / Certifications
- Aptitude Test Score
- Soft Skills Rating
- Extracurricular Activities
- Placement Training
- SSC Marks
- HSC Marks
